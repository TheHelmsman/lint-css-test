# Vue3 sample applicationt to test styles and css linter settings

## Project requirements:

Node 14

```
$ nvm use 14
```

## Project setup

```
$ yarn run install
```

### Compiles and hot-reloads for development

```
$ yarn run serve
```

### Compiles and minifies for production

```
$ yarn run build
```

### Run your unit tests

```
$ yarn run test:unit
```

### Run your end-to-end tests

```
$ yarn run test:e2e
```

### Lints and fixes files

```
$ yarn run lint
$ yarn run lint:fix
```

### SCSS Lints

```
$ yarn run lint:scss
```

### Run your prettier

```
$ yarn run prettier
```

### Run your format = prettier + lint:fix

```
$ yarn run format
```

### Run your cz

```
$ yarn run cz
```

### Run your commit

```
$ yarn run commit
```

### Run your version

```
$ yarn run version
```

### Run your ncu (npm-check-updates)

```
$ yarn run ncu
```
